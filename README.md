Project name: "Breast Cancer Classification"
The Breast Cancer Classification project involves building a machine learning model 
to accurately identify histology images as either benign or malignant. 
This is achieved by training a Convolutional Neural Network (CNN), specifically named CancerNet,
using a large dataset of histology image patches to detect Invasive Ductal Carcinoma (IDC), 
the most common form of breast cancer.
